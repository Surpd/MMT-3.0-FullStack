### 📄 `recommendation_service.py`
| Функция | Аргументы | Описание |
|---|---|---|
| `__init__` | `(self, db, tmdb, session_cache, recs_pool_cache)` | *описание отсутствует* |
| `_calculate_genre_weight` | `(self, count)` | Логарифмическое сглаживание (Log Normalization). Защищает от доминации одного жанра, если юзер лайкнул его 50 раз. |
| `_calculate_recency_bonus` | `(self, release_year)` | Плавное старение (Smooth Recency). Дает бонус свежим фильмам, который плавно тает с годами. |
| `_biased_shuffle` | `(self, items)` | Взвешенный шаффл (Biased Shuffle). Добавляет легкий шум (-0.5 до 0.5) к скору, чтобы перемешать фильмы  со схожим рейтингом, но не утопить реальные шедевры. |
| `_apply_diversity_and_protect_top` | `(self, ranked_candidates)` | Защита ТОП-5 и разбавка остальных (Top-5 Protection + Diversity). |
| `async _get_user_context` | `(self, user_id)` | Собирает профиль: глобальные веса жанров, сессионные якоря и черный список. |
| `async _fetch_candidates_from_tmdb` | `(self, top_genres, recent_liked_ids)` | Генерирует сырой пул кандидатов из 3 источников (TMDB). |
| `_score_candidates` | `(self, candidates, genre_weights, recent_liked_ids, session_data)` | Прогоняет сырых кандидатов через аддитивную математику: Базовый вес + TMDB + Session Bonus/Penalty. |
| `async get_next_movies` | `(self, user_id, cursor)` | Главный метод (входная точка для API).  Отдает 10 фильмов, используя кэш или генерируя новый пул. |
